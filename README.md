## Codebase Structure
- `/src/`: Core database engine, query parsing, and serialization rules.
  - `.infinitum.h` / `infinitum.h`: Core infinite-precision definitions, infinite recursion, and neuromorphic bounds. Establishes the `.infinitum` Top-Level Domain (TLD) as the namespace for Follow-Native programs and "Interstellar-ready Production Systems" in the Canor Infinitum FOSS Ecosystem.
  - `.repositorium.h` / `database.h` / `generator.h`: Database repository core, transactions, generator routines, and Larmor-sync validation hooks. Establishes the `.repositorium` Top-Level Domain (TLD) as the namespace for decentralized programming projects.
  - `arbitrary.h` / `length.h` / `point.h` / `axis.h`: Simple-Arbitraries, Simple-Lengths, axes, and coordinate mapping based on u8 intervals.
  - `atom.h` / `change.h` / `force.h`: Primitives representing ATOM (value), CHANGE (reference), and FORCE (functional) structures.
  - `chain.h` / `process.h` / `markov.h`: Markov processes, Simple-Chains, and state transition graphs.
  - `composite.h` / `composition.h` / `structure.h`: Composites and structures mapping hyper-complex geometries.
  - `tensor.h` / `tensorial.h`: Covariant, contravariant, symmetric, and antisymmetric tensor arrays.
  - `unit.h` / `unital.h` / `uniter.h` / `unity.h` / `expression.h`: Unit types, expressions, and orthonormalization helpers.
  - `unknown.h` / `unbeknownst.h` / `unique.h`: Unique, unknown-symmetric, and unresolved moduli-space slots.
- `/lib/`: Mathematical libraries for surreal numbers, hyper-complex arrays, and Markov models.
- `/data/`: Physical data tables, transaction logs, and persistent state files.
- `/tmp/`: Development cache and temporary files.
